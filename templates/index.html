Even though it was fun to learn these languages I knew that to be able to create something with actual functionality I had to learn a back-end language, so I decided to learn Python as my primary back-end language and Java as my secondary back-end language! After learning the basics to all these languages I decided to start creating projects because that’s the only way I was going to take my learning to the next level.
